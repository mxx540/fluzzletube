## 这东西到底是什么

Fluzzle Tube 是一家美国亚利桑那州的家族企业做的产品。创始人的儿子有次去漂流，觉得用绳子把泳圈绑在一起太麻烦，也不安全，就想能不能做个可以直接扣在一起的浮筒。经过一番折腾，就有了现在这个"会拼图"的泳圈。

说白了，就是在充气浮筒的边缘装了耐用的尼龙扣件，像拼图块一样，"咔嚓"一声就能扣上。想连几个连几个，想分开就分开，不用费劲搞绳子，更不会缠成一团。

## 用起来怎么样

所有型号都标配了两个杯架、两个把手，还有 12 个连接扣。杯架是可伸缩的那种，能放易拉罐、瓶子，甚至保温杯。把手结实，抓着很稳。

充气很方便，有快速充气阀，几分钟就能搞定。放气更快，不到 60 秒就能收起来。这点特别实用，毕竟谁也不想花半天时间收拾东西。

材质用的是 20 号 6P-free 环保乙烯基，不含有害物质。尺寸大约是 34 英寸长 x 30 英寸宽（约 86cm x 76cm），成年人坐着很舒服，不会觉得挤。

每个浮筒重量不到 5 磅（约 2.3 公斤），女生也能轻松拎着走。配了补丁包，万一哪里破了可以自己修补。

## 注意事项

虽然 Fluzzle Tube 设计得挺好，但还是有几点要提醒一下：

1. **不是救生设备**：它只是个娱乐浮筒，不能当救生圈用。官方建议 13 岁以上使用，不会游泳的人一定要穿救生衣。

2. **尺寸偏大**：充气后体积不小，如果车子后备箱小，可能装不下太多个。好在放气后很紧凑，折叠起来不占地方。

3. **连接处需要点力气**：尼龙扣件很结实，但第一次用可能不太习惯怎么扣。多试几次就熟练了。女生或者手劲小的人可能需要稍微费点力。

4. **颜色和款式不一定常备货**：热门颜色经常断货，尤其是夏天旺季。如果看中了某个配色，最好早点下单。
